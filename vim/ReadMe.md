Installation
------------
1. place `.vim`/`.vimrc` to Home direcotry:

* **Linux**:
```
$ git clone https://github.com/Why8n/dotfiles.git
$ cd dotfiles/vim
$ ./bootstrap.sh
```

* **Windows**:
```
$ git clone https://github.com/Why8n/dotfiles.git
$ cd dotfiles\vim
$ bootstrap.bat
```

2. open `vim`/`nvim`,type`:PlugInstall`
